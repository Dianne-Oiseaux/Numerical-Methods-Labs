## Решение СЛАУ

### М. Гаусса с выбором главного элемента по строке

В каждой строке ищется наибольший по модулю элемент. Если наибольший элемент равен 0, то матрица вырождена, единого решения нет. Если наибольший элемент является k-ым в k-ой строке (до него все нулевые), то оставляем его в покое. Если элемент ненулевой и не первый, то меняем их с текущим столбцом. 

Сохраняем, какие столбцы мы меняли местами, это влияет на порядок корней.  
Также сохраняем количество перестановок, они влияют на знак определителя.

`swap_count` - количество перестановок столбцов
`perm` - для отслеживания номеров столбцов, которые перемещали, чтобы вернуть их назад при определении корней.
